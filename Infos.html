Tout d'abord le système d'exploitation

Le système d'exploitation est le programme permettant de faire tourner le système. Malheureusement Windows n'est pas très "privacy friendly"

Windows possède un programme nommé "editeur de registre" qui note toutes les modifications faites sur le sysème. En plus de ça il s'est révélé que Windows est un énorme spyware. Un logiciel espion récoltant énormémant d'informations afin de les revendre à différents tiers.

Pour ça, il existe plusieurs systeme d'exploistation orienté sur la vie privée basé tels que :

Icone Nom Description
Debian Debian est une distribution Linux, composée presque exclusivement de logiciels libres
Tails Tails est un système d'exploitation portable utilisant la mémoire vive pour fonctionner
Qubeos Qubes OS est un système d'exploitation orienté sécurité grâce à l'isolation des applications.
